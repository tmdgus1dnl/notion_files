### 쓰레기 자동으로 버려주는 로봇

**이미지 분류, 자율 주행, 맵핑**

1. 컴퓨터 책상 옆, 작업 테이블 옆 등 원하는 위치에 RC카 배치
2. RC카 위에 박스 설치
3. 박스에 쓰레기를 놓으면 카메라를 통해 쓰레기가 어떤 종류인지 인식
4. 분류된 쓰레기를 맵핑된 위치(예를 들어, 일반쓰레기면 쓰레기통, 패트병이면 재활용 분리수거함 등)로 이동하여 버리기
5. 다시 원래 자리로 복귀

한승현

1. 공간을 매핑 or 맵을 코드로 만들기
    1. 격자 형태로 구성
2. 물건을 받기 위해 특정 장소로 이동
    1. 받는 위치를 입력 받기
        1. 원격 입력(통신) or LCD 터치
    2. 받는 위치를 리스트 or 큐 형태로 저장
    3. 자율 주행
        1. 매핑 데이터 기반으로 중앙에 임의의 선을 그어 주행
        2. map에 거점을 두고 각 거점까지 레이더로 실시간 벽 판단하여 주행
    4. 주행 중 돌발 장애물 회피
        1. 레이더 or 초음파 센서로 감지 가능
            1. 초음파 센서 → 전방
            2. 레이더 → 360도 + 약간 위 아래도 가능
    5. 물건을 스스로 집거나 사람이 올려놓는 방식
        1. 집게 
        2. 리프트
        3. 들리는 바구니
        4. 바구니
3. 받고나서 반납 위치로 이동
    1. 어떤 물건인지 객체 인식
        1. 카메라 전면 or 후면 부착
    2. 물건을 집었는지 판단 후 후 출발
        1. 집게 → 모르겠다.
        2. 바구니 → 무게 or 카메라로 인식
        3. 목적지는 객체 인식 정보 기반으로 판단
    3. 물건을 스스로 놓거나 사람이 놓는 방식
4. 반납 후 Task pop, Task 가 남았다면 위 반복 수행

구진영 (쓰레기 분류)

- 사전 세팅(초기 실행)
    - 매핑(기본적으로 주행→충돌방지 알고리즘 방식으로 주행)
        - 일단 격자 형태 생각 중
    - 쓰레기통 위치 인식 → 맵에 추가
        - 시간 부족 판단 시 사용자가 웹페이지나 앱 통해서 직접 추가하는 방식
    - 기본 순찰 경로 생성(AI한테 맡기거나 알고리즘 생각)
        - 시간 부족 판단 시 순찰 기능 빼거나 벽이랑 일정 거리 두고 벽 따라 주행
- 기본 동작
    - 평상 시 순찰 경로 따라 순찰 → 쓰레기 인식 시 쓰레기 앞까지 주행
        - 시간 부족 판단 시 순찰 빼고 마커를 통해서든 맵 지정이든 대기 위치 지정, 쓰레기를 직접 올려놓는 방법으로
    - 쓰레기 줍기(방식은 집게든 바구니든 정해야 함)
    - 주운 쓰레기 인식 → 종류 분류
    - 분류된 결과 따라 해당 쓰레기통으로 경로 생성, 주행
        - 경로는 다익orA*로, 물체 크기 고려, rc카가 부드럽게 움직이게 하는 방법 생각해야 함
    - 주행 중 장애물 만날 시 충돌방지(알고리즘은 매핑 때 쓴 거 쓰던가 새로 만들던가) → 경로로 복귀
        - 현실적으로 움직이는 장애물 불가, 고정된 장애물, 장애물이 쓰레기일 시 일단 회피하는 방향으로
    - 도착 후 쓰레기통 위치 정확히 맞춰서 버리기(카메라 기반)
- 웹 서버 or 앱
    - 주행 중 카메라 화면 출력
    - 물체 분류 결과
    - 맵 + rc카 현재 위치 출력
    - 쓰레기통 위치 인식해서 자동 맵 생성 안될 시 쓰레기통 위치 지정하는 기능
    - 순찰 없을 시 + 마커 인식 안쓸 시 대기위치 지정하는 기능
- 기타 기능(임베디드), 고려 사항
    - 매핑 + 충돌방지는 기본적으로 라이다 사용, 써보고 추가 센서(초음파) 사용 판단
    - 현재 위치 계산은 라이다 + IMU, 그래도 오차 크면 모터엔코더 사용 고려
    - jetson orin nano, 라파 역할 정해야 함
    - 카메라 위치, 카메라를 주행용, 인식용 나눌지

공재국

- 시연 과정
    - 초기 지점에서 매핑된 위치로 이동
    - 물류(박스)에 쓰여진 글자로 종류 인식
    - 각 종류별 매핑된 공간 중에 알맞은 종류의 공간으로 이동
    - 이동 중 장애물이 나타나면 보이는 면을 벽으로 인식하여 실시간으로 새로운 경로 생성
    - 목적지에 도착하면 물류 내려놓고 다시 복귀
