# 파라미터 조정 가이드

```c
• 테스트하면서 볼 파라미터는 우선 이 정도면 됩니다.

  1. 계속 왼쪽으로 틀어짐

  먼저 이 값만 조정하세요.

  straight_wz_trim_radps: -0.02

  왼쪽으로 틀어지면 더 음수:

  -0.03
  -0.04

  오른쪽으로 틀어지기 시작하면 너무 많이 넣은 겁니다.

  -0.01
  0.0

  2. 오른쪽으로 틀어짐

  straight_wz_trim_radps가 과하거나 부호가 반대일 수 있습니다.

  straight_wz_trim_radps: -0.01

  그래도 오른쪽이면:

  straight_wz_trim_radps: 0.0

  만약 원래부터 오른쪽으로 틀어지는 상황이면 양수도 테스트합니다.

  straight_wz_trim_radps: 0.01

  3. 뒷다리가 아직 바닥을 끎

  rear lift를 더 올립니다.

  lift_z_mm_per_leg: [13.0, 13.0, 24.0, 24.0]

  그래도 끌면:

  lift_z_mm_per_leg: [15.0, 15.0, 26.0, 26.0]

  단, 너무 올리면 뒷다리 swing 때 몸이 흔들리거나 서보 토크가 더 부족해질 수 있습니다.

  4. 뒷다리를 들 때 몸이 뒤로 주저앉음

  보행을 느리게 합니다.

  gait_period_sec: 1.5
  vx_max_mps: 0.12

  그리고 발 기본 위치를 조금 조정합니다.

  default_foot_x_mm_per_leg: [-25.0, -25.0, -25.0, -25.0]

  -20보다 더 음수는 몸 기준 발을 더 뒤로 두는 쪽입니다. 너무 가면 앞다리 자세가 불편해질 수 있으니 5mm 단위로만 바꾸세요.

  5. 발은 들리는데 보폭이 답답하고 너무 느림

  안정적이면 속도만 조금 올립니다.

  vx_max_mps: 0.18

  그래도 안정적이면:

  gait_period_sec: 1.3

  한 번에 둘 다 빠르게 바꾸지는 마세요.

  6. 몸이 좌우로 많이 흔들림

  보폭과 lift를 줄입니다.

  max_stride_x_mm: 60.0
  lift_z_mm_per_leg: [12.0, 12.0, 20.0, 20.0]

  또는 duty factor를 조금 올려 접지 시간을 늘립니다.

  duty_factor: 0.62

  7. 앞다리가 너무 많이 부담받거나 앞쪽으로 고꾸라지는 느낌

  default_foot_x_mm_per_leg를 덜 음수로 되돌립니다.

  default_foot_x_mm_per_leg: [-15.0, -15.0, -15.0, -15.0]

  또는 전후 다리를 다르게 둡니다.

  default_foot_x_mm_per_leg: [-15.0, -15.0, -20.0, -20.0]

  추천 테스트 순서

  3. 안정화되면 vx_max_mps와 gait_period_sec를 조금씩 되돌려 속도 회복
  
 다리 드는거 적용할 만한 파라미터 lift_z_mm_per_leg: [18.0, 13.0, 30.0, 26.0]
```
