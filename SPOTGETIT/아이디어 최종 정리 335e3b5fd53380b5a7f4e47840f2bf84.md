# 아이디어 최종 정리

1. **RTScope**: 
    
    FreeRTOS에서 태스크 전환, IPC, 성능 지표를 실시간으로 추적해 UART/DMA로 PC 터미널에 보여주는 오픈소스 트레이싱 라이브러리.
    
2. **블록 코딩**: 
    
    코드보다 시각적 블록 조합으로 제어 로직을 구성하는 교육형·입문형 임베디드/로봇 프로젝트 아이디어.
    
3. **밸런싱 로봇**: 
    
    IMU와 실시간 제어 루프를 이용해 2륜 자세를 안정적으로 유지하는 제어 중심 프로젝트 아이디어.
    
4. **와이파이 CSI**: 
    
    Wi-Fi 채널 상태 정보를 활용해 움직임이나 존재를 감지하는 비전 비의존 센싱 프로젝트 아이디어.
    
5. **ROS2-FreeRTOS 기반 범용 실시간 모바일 로봇 하부 제어 패키지**: 
    
    상위 ROS2 알고리즘과 하위 MCU 제어를 표준 메시지로 연결해, 어떤 보드에도 붙일 수 있는 Plug & Play형 실시간 하부 제어 패키지를 만드는 프로젝트.
    
6. **RTOS 기반 무인 플랫폼 하위 안전 제어기 ECU 플랫폼**: 
    
    UGV/UAV 전체가 아니라, 상위 명령 수신·센서 수집·상태 판단·제어 출력·안전 전이·진단 복구를 담당하는 공통 하위 ECU 플랫폼 + 관제 시스템을 만드는 프로젝트.
    
7. **다중 노드 분산 제어 시스템**: 
    
    Sensor Node, Master Control Node, Actuator Node를 나눠 CAN/UART로 연결하고, 센서·판단·출력을 분산시켜 하나의 ECU 네트워크처럼 동작하게 만드는 구조형 프로젝트.
    
8. **미래형 전차(★★★탱크, 그런데 “시즈모드”가, 가능할지도?!★★★**) : 
    
    원격 주행, 타겟 락온, 상탑 추적, 카메라 원격 송출까지 포함한 미래형 무인 전차 콘셉트 프로젝트.
    
    궤도 차량 기반에 자동 타겟 기능과 시즈모드 개념을 얹은 전투형 로봇 아이디어.
    
9. **우러 무인 로봇 참고형 아이디어(**[https://www.youtube.com/watch?v=F-Jff0dkrLU](https://www.youtube.com/watch?v=F-Jff0dkrLU)): 
    
    특정 무인 로봇 사례를 레퍼런스로 삼아 방산형 무인 플랫폼 방향을 잡는 참고형 주제 아이디어.
    
10. **이기종 시스템 간 Zero-Latency 통신 프레임워크 ★**:
    
    SBC와 MCU 사이의 통신 병목을 DMA, 경량 프로토콜, 커스텀 트랜스포트로 줄여 더 낮은 지연과 높은 결정성을 노리는 미들웨어 프로젝트
    
    6번 + 7번 이랑 연계 가능 → 투트랙
    
11. **로봇 제어 검증을 위한 디지털 트윈 기반 HILS 환경**:
    
    가상 시뮬레이터와 실제 MCU 펌웨어를 실시간 동기화해, 하드웨어 없이도 제어 로직을 검증하는 테스트베드 프로젝트.
