# 파인튜닝 데이터셋 수집 계획표

상태: VISION

> 작성일 : 2026-04-21
프로젝트 : SPOT Get IT
목적 : OAK-D Lite 실제 입력 기반 파인튜닝 데이터 수집
> 

---

## 1. 수집 목적

- OAK-D Lite RGB 카메라의 실제 입력 환경에 맞는 데이터 확보
- 팀원 복장 다양성 반영
- 파인튜닝을 통한 실제 환경 탐지 성능 향상

---

## 2. 수집 환경

| 항목 | 내용 |
| --- | --- |
| 카메라 | OAK-D Lite (CAM_A, RGB) |
| 촬영 해상도 | 1080P (1920x1080) |
| 전처리 | 촬영 후 크롭 → 480x480 리사이즈 |
| 촬영 장소 | 실내 (천장 형광등 조명) |
| 배경 변화 | 카메라 각도를 틀어 배경 다양화 |
| 피사체 위치 | 항상 프레임 중앙 고정 |

---

## 3. 피사체 구성

| 항목 | 내용 |
| --- | --- |
| 인원 | 팀원 6명 (A, B, C, D, E, F) |
| 인당 일일 촬영 수 | 10장 |
| 일일 총 촬영 수 | 60장 |
| 수집 주기 | 매일 아침 (월~금) |

---

## 4. 인당 10장 촬영 기준

피사체 위치는 중앙 고정, 피사체 각도를 변경하여 다양한 시점 확보

| 장 | 거리 | 각도 |
| --- | --- | --- |
| 1 | 근거리 (1~2m) | 좌 90도 |
| 2 | 근거리 (1~2m) | 좌 45도 |
| 3 | 근거리 (1~2m) | 정면 (0도) |
| 4 | 근거리 (1~2m) | 우 45도 |
| 5 | 근거리 (1~2m) | 우 90도 |
| 6 | 중거리 (3~5m) | 좌 90도 |
| 7 | 중거리 (3~5m) | 좌 45도 |
| 8 | 중거리 (3~5m) | 정면 (0도) |
| 9 | 중거리 (3~5m) | 우 45도 |
| 10 | 중거리 (3~5m) | 우 90도 |

## 6. 주간 스케줄 (1주차 기준)

자세 6가지와 인원 6명을 1:1 매칭하여 하루에 전체 자세가 커버되도록 운영한다.
다음 주차부터는 인원을 한 칸씩 이동하여 순환한다.

| 자세 | 월 | 화 | 수 | 목 | 금 |
| --- | --- | --- | --- | --- | --- |
| 선 자세 | C | D | E | A | B |
| 의자 착석 | D | E | F | B | C |
| 바닥 착석 | E | F | A | C | D |
| 눕기 | F | A | B | D | E |
| 의자 착성 (상체만 보이게) | A | B | C | E | F |
| 옆으로 눕기 | B | C | D | F | A |

| A | 배 |
| --- | --- |
| B | 방 |
| C | 김 |
| D | 구 |

## 7. 라벨링 계획

| 항목 | 내용 |
| --- | --- |
| 라벨링 도구 | Roboflow |
| 담당 | 본인 직접 수동 라벨링 |
| 클래스 | person (단일) |
| 라벨링 주기 | 당일 촬영 후 |

---

## 8. 전처리 파이프라인

```
OAK-D Lite 촬영 (1080P)
        ↓
Roboflow 업로드
        ↓
수동 라벨링 (person bbox)
        ↓
크롭 및 480x480 리사이즈
        ↓
기존 데이터셋에 병합
        ↓
파인튜닝 학습
```

---

## 9. 통제 요소 요약

| 항목 | 통제 방식 |
| --- | --- |
| 조명 | 실내 천장 형광등 고정 |
| 피사체 위치 | 프레임 중앙 고정 |
| 배경 | 카메라 각도 변경으로 자연스럽게 다양화 |
| 좌/우 위치 다양성 | 1080P 크롭 위치 조정으로 확보 |
| 거리 | 근거리(1~2m), 중거리(3~5m) 2단계 고정 |
| 각도 | 좌90, 좌45, 정면, 우45, 우90 5단계 고정 |
| 복장 | 매일 아침 자연스럽게 다양화 |
