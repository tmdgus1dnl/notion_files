# Spot Micro 전원부 설계

## 1. 개요

12개의 Feetech STS3215 서보, Jetson Orin Nano 8GB, STM32F446RE Nucleo-64를 안정적으로 구동하기 위한 전원 분배 시스템. **Jetson 전원을 서보/STM32 라인과 분리하여 노이즈 영향을 최소화**하는 것을 핵심 설계 원칙으로 함.

---

## 2. 전원 소스

| 용도 | 사양 | 비고 |
| --- | --- | --- |
| 데모/실전 | 3S LiPo 5400mAh (40C 이상) | XT60, 셀 체커 장착 |
| 벤치/개발 | Mean Well LRS-350-12 (12V 29A) | 회로 디버깅, 장시간 시험 |
| 전환 방식 | XT60 커넥터 물리적 교체 | 다이오드 OR-ing 미사용 |

7200mAh(139×47×37mm, 435g)는 두께 37mm가 프레임 STL과 호환되는지 확인 후 추가 검토.

---

## 3. 전원 분배 토폴로지

```
 [3S LiPo or LRS-350-12]
            │
         XT60 (On/Off 역할 겸용)
            │
       ATO 30A 퓨즈 (인라인)
            │
       XT60 Y-Splitter
        ┌───┴───────────────────┐
        │ 서보 + STM32 분기       │ Jetson 분기
        │                        │
        ▼                        ▼
    [PDB + 3300µF 벌크캡]    [Buck-Boost
        │                     12V → 19V]
   ┌────┼────┬────┬────┐         │
   ▼    ▼    ▼    ▼    ▼         ▼
  Leg  Leg  Leg  Leg  [5V BEC]   Jetson Orin Nano
  FL   FR   RL   RR    │         (배럴잭)
   │    │    │    │    ▼
  3×   3×   3×   3×   STM32 Nucleo
       STS3215         (5V 핀)
```

**Y-Splitter 분리 이유 (면접 답변용 3가지)**:

1. **전압 도메인 분리**: Jetson Orin Nano는 19V 배럴잭 입력이 필요하므로 별도 Buck-Boost 컨버터가 필수. PDB의 12V Aux 출력은 일반적으로 500mA 한계로 Jetson 피크(약 2A) 구동 불가
2. **노이즈 격리**: 12개 서보가 동시 스위칭할 때 발생하는 전류 리플이 Jetson 전원 평면을 통해 전파되는 것을 물리적으로 차단. STM32는 BEC의 step-down 컨버터가 1차 필터 역할
3. **고장 전파 차단**: Jetson 측 단락이 서보/STM32 측에 영향 없음 (반대 방향도 마찬가지)

## 4. 주요 부품 사양

### 4.1 퓨즈

| 항목 | 값 |
| --- | --- |
| 형식 | ATO 32V 30A Fast-blow (동우LB) |
| 차단 용량 (AIC) | 1,000A |
| 위치 | 배터리 XT60 직후 10cm 이내 |
| 착탈 | 퓨즈 홀더 양단 XT60으로 배터리 측에 체결 |

**30A 선정 근거**: 12 서보 × 2A 스톨 = 24A 피크 + Jetson/STM32 약 2A → 연속 약 20A, 피크 25~26A. 20% 마진으로 30A.

### 4.2 PDB (Power Distribution Board)

- 입력: 12V (배터리 또는 SMPS)
- 출력 1: 4 채널 × 다리당 STS3215 3개 (총 12개)
- 출력 2: 5V BEC 분기 → STM32
- 벌크 커패시터: **3300µF 전해** (저ESR, 16V 이상)
- 배선: 14~16 AWG, Star topology

### 4.3 Jetson 전원: Buck-Boost (LTC3780 기반)

| 항목 | 값 |
| --- | --- |
| 입력 | 12V (3S LiPo 만충 12.6V ~ 방전 컷오프 10.2V) |
| 출력 | **19V 고정** |
| 출력 전류 | 연속 3.5A 이상 |
| 입출력 캡 | 470µF × 2 (RVT 전해) |
| 대상 | Jetson Orin Nano 배럴잭 |

**19V 출력 선정 근거**:

- Jetson 입력 범위 7~20V 중 공식 어댑터 사양과 일치
- Buck-Boost 입력(12V)이 출력(19V)보다 항상 낮아 **부스트 모드 고정** → Buck↔Boost 전환 구간(약 11~13V)을 회피하여 전원 안정성 확보
- LiPo 방전 시 12.6V→10.2V로 변동해도 출력 19V 유지

### 4.4 STM32 전원: BEC (LM2596)

- 입력: **PDB 12V 분기**
- 출력: 5V / 1A
- 대상: STM32 Nucleo-64 (5V 핀)
- BNO055 IMU 3.3V는 STM32 온보드 LDO에서 공급

**PDB에서 BEC로 분기한 이유**:

- 서보와 STM32 모두 동일한 12V 도메인을 공유하므로 추가 분배기 불필요
- BEC의 step-down 스위칭이 서보 측 리플에 대한 1차 필터 역할
- 만능기판에 BEC 모듈을 두고 STM32 5V 핀에 직결

## 5. 전류 예산

| 부하 | 연속 | 피크 | 비고 |
| --- | --- | --- | --- |
| 시스템 합계 | ~20A | ~25A | 3S 12V 기준 |
| 서보 12개 | 15~18A | 22A | 보행 중 평균 1.5A/서보, 스톨 2A |
| Jetson Orin Nano | 1.3A (12V 환산) | 2A | MAXN_SUPER 모드 |
| STM32 + 센서 | <0.5A | <0.5A | BNO055, 통신 버퍼 포함 |

**LiPo C-Rate 마진**: 5400mAh × 40C = 216A 방전 가능 → 25A 대비 **8배 이상 마진**.

---

## 6. 접지 설계 (Star Grounding)

모든 GND는 **배터리 (-) 단자 근처 단일 지점**에서 합류.

```
배터리 (-) ─┬─ 서보 GND (PDB 경유)
            ├─ STM32 GND (BEC → PDB 경유)
            ├─ Jetson GND (Buck-Boost 경유)
            └─ 만능기판 공통 GND 버스 (센서, 신호선)
```

신호선 GND와 전원 GND를 **단일 지점에서만 연결**하여 그라운드 루프 방지.

---

## 7. LiPo 운용 안전

- **셀 체커**: 3S 밸런스 커넥터에 장착, 셀당 3.3V 알람
- **저전압 보호**: Buck-Boost UVLO 약 10.2V (셀당 3.4V) 설정
- **단락 보호**: ATO 30A 퓨즈가 1차 보호
- **충전**: XT60 분리 후 전용 밸런스 충전기 사용
- **보관**: Storage 전압 (셀당 3.8V) 유지, LiPo 안전 가방 내 보관

## 8. 설계 결정 요약 (면접 답변용)

| 결정 | 근거 |
| --- | --- |
| Y-Split 토폴로지 | Jetson 19V 도메인 분리 + 서보 노이즈로부터 Jetson 격리 |
| STM32를 PDB→BEC로 연결 | 동일 12V 도메인 공유, BEC가 1차 노이즈 필터 역할 |
| Buck-Boost 19V 출력 | Buck↔Boost 전환 구간 회피, Jetson 공식 어댑터 사양 일치 |
| ATO 30A 퓨즈 | 피크 25A + 20% 마진, 소형 LiPo 로봇 업계 표준 |
| 10 AWG 배선 (메인) | 30A 퓨즈 정격에 대응하는 최소 단면적 |
| 로커 스위치 미사용 | XT60 체결/해제로 충분, 부품 추가 시 접점 저항/고장점 증가 |
| XT90 안티스파크 미사용 | 470µF 정도 캡 용량에서는 인러시 전류 미미 |
| LRS-350-12 벤치 PSU | 디버깅 시 LiPo 사이클 소모 방지, 안정 전원 확보 |

## 9. 검증 체크리스트

- [ ]  Buck-Boost 무부하 출력 19.0V ± 0.1V
- [ ]  BEC 무부하 출력 5.0V ± 0.1V
- [ ]  퓨즈 홀더 접촉 저항 5mΩ 이하
- [ ]  배터리 만충 무부하 12.6V 확인
- [ ]  모든 GND가 단일 지점에서 합류 확인 (멀티미터로 도통)
- [ ]  셀 체커 알람 동작 확인 (3.3V 임계값)
- [ ]  첫 부하 인가 시 퓨즈 홀더/배선 발열 체크 (5분, 적외선 온도계)
- [ ]  Jetson `tegrastats` 모니터링하며 MAXN_SUPER 모드 안정성 확인
- [ ]  서보 12개 동시 스톨 시 STM32 5V 라인 리플 측정 (오실로스코프, 100mV pk-pk 이하 목표)
- [ ]  동일 조건에서 Jetson 19V 라인 리플 측정 (Y-Split 격리 효과 검증)

- **E-Stop 스위치**: 필드 테스트 단계에서 비상정지 회로 검토

      │         │              │

      │        + 외부 470µF     + 외부 100µF
      │        + 10µF MLCC      + 10µF MLCC

    ## 3. 극성 규칙 (매우 중요)
    
    ### 전해캡 극성 구별
    
    **몸체의 흰 줄 = (−) 음극**
    
    ```
          ┌───┬─┐
          │   │▓│  ← 흰 세로줄 = (−)
          │   │▓│
          └───┴─┘
             ↑
        반대쪽 = (+)
    ```
    
    **리드 길이로도 구별 가능**:
    
    - 긴 리드 = (+)
    - 짧은 리드 = (−)
    
    ### 와이어 색상 규칙
    
    - **빨강 = (+)** 양극
    - **검정 = (−)** 음극
    
    ### 방향 틀렸을 때 결과
    
    - 5V 이하: 서서히 부풀어오름, 누액
    - 10V 이상: **즉시 폭발**, 파편 + 전해액 분출 → 안전 고글 필수
    
    ### 납땜 전 체크 절차
    
    ```
    1. 캡 흰 줄 위치 확인 → (−)
    2. 자르기 전에 (+) 리드에 유성 매직으로 점 찍기
    3. 리드 5mm 남기고 자름
    4. 납땜 전 마지막으로 극성 재확인
    5. 납땜 후 멀티미터 저항 모드로 쇼트 체크
    ```

    ## 4. 작업 단계
    
    ### 단계 0: 벅부스트 출력 전압 세팅 (반드시 먼저)
    
    **로봇에 연결하기 전에** 벅부스트 2개의 출력 전압을 세팅.
    
    ```
    1. 벅부스트 입력에 SMPS(12V) 또는 배터리 연결
    2. 출력은 아무것도 연결 안 함 (빈 상태)
    3. 멀티미터 DC 전압 모드로 OUT+ / OUT- 측정
    4. 파란 볼륨 돌려서 정확히 5.0V 맞춤 (±0.05V)
    5. 둘 다 5V 확인 → 전원 OFF
    ```
    
    5V 넘으면 Jetson/STM32 파손 가능. 반드시 먼저 확인.
    
    ### 단계 1: 메인 캡 뱅크 (XT60 연장선 활용)
    
    **준비**: XT60 연장 케이블 1개 (암/수 한쪽씩)
    
    **작업**:
    
    ```
    1. 연장선 중간 5cm 지점에서 빨강/검정 와이어 피복 각 5mm 벗김
    2. 2200µF #1 준비:
       - 흰 줄 확인, (+) 쪽 매직 표시
       - 리드 5mm 남기고 자름
    3. 2200µF #1 (+) 리드를 빨간 와이어 벗긴 부분에 납땜
    4. 2200µF #1 (−) 리드를 검정 와이어 벗긴 부분에 납땜
    5. 같은 위치 근처에 2200µF #2도 동일하게 병렬 납땜
    6. 10µF MLCC도 같은 위치에 병렬 납땜 (극성 무관)
    7. 열수축 튜브로 전체 덮음
    ```
    
    **결과물**:
    
    ```
    배터리 XT60 ━━━━━[캡3개 병렬]━━━━━ 로봇 XT60
                          │
                     열수축 튜브
    ```
    
    ### 단계 2: 벅부스트 1 (Jetson용) 입력단에 캡 추가
    
    **준비**: 1000µF 25V 전해캡 1개
    
    **작업**:
    
    ```
    1. 벅부스트 1의 IN+ / IN- 단자 확인 (보드 실크 표시)
    2. 1000µF 캡 극성 확인, 리드 5mm로 자름
    3. 캡 (+) 리드를 IN+ 패드에 납땜
    4. 캡 (−) 리드를 IN- 패드에 납땜
    5. 캡이 보드 위에 세워지는 형태가 됨
    6. 열수축 튜브로 리드 부분만 절연
    ```
    
    **공간이 부족하면**: 와이어 피복 벗긴 부분에 병렬로 달아도 OK. 다만 벅부스트 패드에서 1cm 이내 거리여야 효과 있음.
    
    ### 단계 3: 벅부스트 1 (Jetson용) 출력단에 캡 추가
    
    **준비**: 470µF 16V 전해캡 1개, 10µF MLCC 1개
    
    **작업**:
    
    ```
    1. 벅부스트 1의 OUT+ / OUT- 단자 확인
    2. 470µF 극성 확인, 리드 5mm로 자름
    3. 470µF (+) 리드 → OUT+ 패드
    4. 470µF (−) 리드 → OUT- 패드
    5. 10µF MLCC도 같은 위치에 병렬 납땜 (극성 무관)
    6. 열수축 튜브로 절연
    ```
    
    ### 단계 4: 벅부스트 2 (STM32용) 입출력단에 캡 추가
    
    **단계 2, 3과 동일한 방식**으로:
    
    - 입력단: 1000µF 25V
    - 출력단: 100µF 16V + 10µF MLCC
    
    ### 단계 5: 서보 레일 로컬 캡 (각 다리 4곳)
    
    **목적**: 서보 12개의 순간 스파이크와 PWM 노이즈를 각 다리 근처에서 로컬 흡수. 메인 캡 뱅크까지 역전파되는 것을 차단.
    
    **준비**: 470µF 16V 전해캡 4개, 10µF MLCC 4개
    
    **작업 위치**: 각 다리에서 서보 3개(어깨/허벅지/종아리)가 만나는 **전원 분배 지점**. 보통 어깨 서보 근처.
    
    **작업 (한 다리당 반복, 총 4회)**:
    
    ```
    1. 해당 다리의 서보 전원 분배 와이어 찾음
       - 서보 3개가 공통으로 쓰는 빨강/검정 배선
       - 어깨 서보 진입부가 가장 적합
    
    2. 분배 와이어 중간 5mm 피복 벗김
       (빨강/검정 각각)
    
    3. 470µF 극성 확인
       - 흰 줄 = (−)
       - (+) 리드에 유성 매직 표시
       - 리드 5mm 남기고 자름
    
    4. 납땜
       - 470µF (+) 리드 → 빨간 와이어 벗긴 부분
       - 470µF (−) 리드 → 검정 와이어 벗긴 부분
    
    5. 10µF MLCC도 같은 위치에 병렬 납땜
       (극성 무관)
    
    6. 열수축 튜브로 캡과 납땜 부위 전체 절연
    
    7. 나머지 3개 다리에도 동일 작업
    ```
    
    **물리적 배치**:
    
    ```
    [다리 배선]
    
    배터리 레일 ━━━━━┓
                     ┃
                     ┣━━━━━[어깨 서보]  ← 이 근처에 캡 부착
                     ┃         │
                     ┃   [470µF + 10µF]
                     ┃         │
                     ┣━━━━━[허벅지 서보]
                     ┃
                     ┗━━━━━[종아리 서보]
    ```
    
    **주의사항**:
    
    - 캡 리드는 **5mm 이내로 짧게** (인덕턴스 최소화)
    - 캡이 다리 움직임을 방해하지 않는 위치에
    - 진동에 의한 단락 방지를 위해 글루건으로 고정 가능
    - 4개 다리 모두 같은 위치/방법으로 통일 (유지보수 용이)
    
    **효과**:
    
    - 서보 스파이크 전압 강하 약 50% 감소
    - 배터리 레일 노이즈 감소 → Jetson/STM32 안정성 향상
    - 서보 간 상호 간섭 감소 → RL 정책 학습 안정화

    ## 5. 벅부스트 → 부하 연결
    
    ### Jetson Orin Nano 연결
    
    **배럴잭 커넥터 납땜**:
    
    ```
    벅부스트 1 OUT+ ──[빨강 18AWG]──→ 배럴잭 중심 핀 (+)
    벅부스트 1 OUT- ──[검정 18AWG]──→ 배럴잭 외부 배럴 (-)
    ```
    
    배럴잭: "DC plug 5.5×2.1mm center positive" 구매. Jetson Orin Nano DC 잭에 꽂힘.
    
    **극성 확인**: Jetson DC 잭은 **center positive** (중심이 +). 거꾸로 연결 시 Jetson 즉시 파손.
    
    ### STM32 Nucleo-F446RE 연결
    
    **중요: JP5 점퍼 먼저 변경**
    
    ```
    1. Nucleo 보드 전원 OFF, USB 분리
    2. JP5 점퍼 위치 확인 (User Manual UM1724 참조)
    3. 점퍼 캡을 U5V → E5V 위치로 이동
    4. 이렇게 안 하면 외부 5V와 USB 5V 충돌 → ST-Link 파손 가능
    ```
    
    **E5V 핀에 연결**:
    
    ```
    벅부스트 2 OUT+ ──[빨강 20AWG]──→ Nucleo E5V 핀 (CN7 핀 6)
    벅부스트 2 OUT- ──[검정 20AWG]──→ Nucleo GND 핀 (아무 GND나 OK)
    ```
    
    Nucleo 내장 LDO가 자동으로 3.3V 생성해서 MCU에 공급.

    ## 6. 전원 인가 순서 (최초 테스트)
    
    **반드시 이 순서로**:
    
    ```
    1. 모든 납땜 부위 시각 검사 (쇼트 여부)
    2. 멀티미터 저항 측정 모드로 V+/V- 간 저항 체크
       - 0Ω = 쇼트, 전원 넣지 마
       - 처음 수십 Ω에서 kΩ으로 상승 = 정상 (캡 충전 중)
    3. SMPS로 먼저 테스트 (배터리 말고):
       - 12V/20A SMPS를 XT60에 연결
       - 1분간 무부하 idle, 발열 확인
       - 벅부스트 출력 전압 재측정 (5V 유지되는지)
    4. 이상 없으면 Jetson DC잭 연결, 부팅 확인
    5. 다음 STM32 연결, LED 점등 확인
    6. 둘 다 정상이면 배터리로 교체해서 동일 테스트
    ```
    
    **첫 배터리 연결 시 스파크 주의**: 대용량 캡이 있어서 순간 "팟!" 소리 남. 정상이지만 눈 보호 필요. Anti-spark 스위치 있으면 그걸로 완화.

    ## 7. 이상 징후 체크리스트
    
    연결 후 이런 증상 나오면 바로 전원 차단:
    
    | 증상 | 원인 | 대응 |
    | --- | --- | --- |
    | 캡 부풀어오름 | 극성 반대 | 즉시 차단, 캡 교체 |
    | 캡 뜨거움 (60°C+) | 극성 반대 or 과전압 | 즉시 차단 |
    | 벅부스트 연기 | 쇼트 or 과부하 | 즉시 차단 |
    | 벅부스트 출력 0V | 내부 파손 | 벅부스트 교체 |
    | 출력 전압 이상 (4V 또는 6V) | 볼륨 틀어짐 | 재조정 |
    | Jetson 부팅 실패 | 전압 부족 or 전류 부족 | 벅부스트 용량 확인 |
    | Jetson 반복 리셋 | 캡 용량 부족 or 벅부스트 노이즈 | 출력단 캡 증설 |
    | STM32 LED 안 켜짐 | JP5 설정 or 극성 | 점퍼 및 배선 확인 |

    ## 8. 물리적 배치
    
    배터리 → 메인 캡 뱅크 → 로봇 본체 진입 위치에 배치:
    
    ```
    [로봇 몸체 상단]
    ┌─────────────────────┐
    │  [배터리 마운트]    │
    │  - 벨크로 + 스트랩  │
    ├─────────────────────┤
    │  [XT60 메인 스위치] │ (선택)
    ├─────────────────────┤
    │  [메인 캡 뱅크]     │
    ├─────────────────────┤
    │  ┌────┐  ┌────┐    │
    │  │벅1 │  │벅2 │    │
    │  │J.슨│  │STM │    │
    │  └────┘  └────┘    │
    ├─────────────────────┤
    │  [Jetson Orin Nano] │
    ├─────────────────────┤
    │  [STM32 Nucleo]     │
    └─────────────────────┘
    ```
    
    **고정 방법**:
    
    - 벅부스트: 양면테이프 + 케이블 타이
    - 캡 뱅크: 3D 프린팅 케이스 or 열수축 튜브로 보호 후 벨크로
    - 와이어: 스파이럴 랩 또는 케이블 타이로 다발로 정리
    
    **주의**: 벅부스트는 발열이 있으니 Jetson 근처는 피하고 통풍 잘 되는 위치에 배치.

    ## 9. 추가 권장 (선택)
    
    ### LVC 알람 (저전압 보호)
    
    - 제품: "3S LiPo voltage alarm" 검색 (₩3,000~5,000)
    - 연결: 배터리 밸런스 커넥터 (JST-XH 4핀)에 꽂음
    - 셀 전압 3.3V 이하로 떨어지면 부저 알람
    - LiPo 과방전 방지용, 강력 권장
    
    ### Anti-Spark 스위치
    
    - 제품: "XT60 anti-spark switch 60A+" 검색 (₩5,000~10,000)
    - 대용량 캡 뱅크 연결 시 스파크 감소
    - 프리차지 → 메인 연결 2단계 동작
    - XT60 커넥터 마모 방지
    
    ### 전압 모니터 ADC
    
    - STM32 ADC로 배터리 전압 실시간 감시
    - 분압 저항 10k:1k (11.1V → 1.01V)
    - 9.3V 이하 시 RL 정책에 저전력 모드 트리거
    - 방산 면접 어필 포인트 (fault-tolerant 설계)

    ## 11. 작업 순서 요약
    
    ```
    1. 부품 주문 (전해캡 + MLCC + XT60 연장선)
    2. 벅부스트 2개 출력 전압 5V로 세팅 (무부하 상태)
    3. XT60 연장선 중간에 메인 캡 뱅크 납땜
    4. 벅부스트 1 입출력단에 캡 추가 (Jetson용)
    5. 벅부스트 2 입출력단에 캡 추가 (STM32용)
    6. **각 다리 서보 레일에 로컬 캡 납땜 (4곳)**
    7. Nucleo JP5 점퍼 E5V로 변경
    8. 벅부스트 → Jetson DC잭, STM32 E5V 핀 배선
    9. SMPS로 무부하 테스트 (1분)
    10. Jetson, STM32 연결 후 부팅 확인
    11. 배터리로 교체, LVC 알람 달기
    12. 10분 idle 테스트
    13. 완료
    ```
    
    작업 시간 예상: **처음이면 4~5시간**, 익숙해지면 1.5시간 이내.
