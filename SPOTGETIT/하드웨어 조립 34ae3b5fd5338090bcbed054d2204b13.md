# 하드웨어 조립

- STS3215 + URT-2
    
    # STS3215 조립 프로세스
    
    ## 핵심 원칙
    
    1. ID 할당 전엔 절대 2개 이상 동시 연결 금지 (전부 기본 ID=1 → 충돌)
    2. 조립 전에 Zero 세팅 (조립 후 틀어지면 분해해야 함)
    3. 한 다리 완성 → 해당 다리 동작 확인 → 다음 다리
    
    ---
    
    ## 프로세스
    
    ### 1. URT-2 단일 서보 연결 확인
    
    - URT-2 SCS power 터미널에 **9V 2A 어댑터** (12V 금지, 터미널 정격 6~9V)
    - SCS 3핀 포트에 서보 1개
    - FD Software → COM 포트 → Baudrate **1000000** → Search → ID 1 검출 확인
    - 안 잡히면: 극성 / 115200 재시도 / SMS 포트 오접속 확인
    
    ### 2. ID 할당 (12개, 1개씩 교체하며)
    
    매핑:
    
    | 다리 | Hip | Thigh | Knee |
    | --- | --- | --- | --- |
    | FR | 1 | 2 | 3 |
    | FL | 4 | 5 | 6 |
    | RR | 7 | 8 | 9 |
    | RL | 10 | 11 | 12 |
    
    절차: Unlock → ID 변경 → Save → Lock → 라벨 부착 → 다음 서보
    
    ### 3. Zero 캘리브레이션 (12개 전부)
    
    - 각 서보 샤프트 중립 방향 잡은 상태에서 FD의 "Middle Point" 버튼 클릭
    - Goal 2048 명령 시 움직임 없으면 성공
    
    ### 4. 개별 스윕 검수
    
    - 0 → 2048 → 4095 왕복
    - Position/Load/Temp 피드백 정상
    - 불량 있으면 조립 전에 교환
    
    ### 5. 다리 조립 (한 다리씩, 총 4회 반복)
    
    각 다리마다:
    
    1. Hip 서보 Goal 2048 상태 → 프레임에 장착, 혼 방향 "수직 아래"
    2. Hip 혼 → 상부 브라켓
    3. Thigh 서보 (2048) → 브라켓에 장착
    4. Thigh 혼 → 대퇴 링크
    5. Knee 서보 (2048) → 대퇴 끝에 장착
    6. Knee 혼 → 경골 링크 + 발끝
    7. 3서보 데이지 체인 (서보 간 포트-포트)
    8. URT-2로 해당 다리 3서보 동작 확인 후 다음 다리
    
    ### 6. 전기 통합
    
    - **PDB 스타토폴로지**: 3S LiPo → 4개 다리 버스 분배 + Jetson 벅부스트
    - 각 다리 버스에 **디커플링 캡** (1000µF 전해 + 0.1µF 세라믹)
    - **74HC126 버퍼 4채널** → STM32 UART 4포트 (다리당 1개 UART)
    - **공통 그라운드**: STM32 / 버퍼 / 서보 버스 / Jetson / IMU 전부 한 점
    - BNO055 IMU 연결
    
    ### 7. 전원 투입 순서
    
    LiPo 밸런스 체크 → STM32·Jetson 로직 먼저 → PDB ON → 5초 대기 → Ping 12개
    
    ### 8. 통합 구동 테스트
    
    1. 전체 Ping (12/12 응답)
    2. 개별 서보 스윕
    3. 다리 단위 포즈 (서기/앉기)
    4. 4다리 Standing Pose
