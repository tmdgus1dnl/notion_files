# 디버거 & 테스터

# 디버거

```bash
valgrind --tool=helgrind  → 동기화 버그 탐지
valgrind --tool=memcheck  → 메모리 오염 탐지
AddressSanitizer (ASan)   → 컴파일 옵션으로 추가
ThreadSanitizer (TSan)    → Race condition 탐지

gcc -fsanitize=thread -g  → TSan 켜기
```

```bash
iperf3   →  무선 구간 처리량/지연 측정
tcpdump  →  실제 패킷 캡처해서 손실률 확인
ping     →  기본 지연 측정
```

# 테스터

### 1. Mock 프로세스 직접 만들기 (가장 현실적)

```c
// fake_receiver.c
// 진짜 Jetson 없어도 공유 메모리에 가짜 데이터 주입
while (1) {
    wrlock();
    shm->odom[0] += 0.1;          // 가짜 좌표
    shm->lidar_size = fill_fake_lidar(shm->lidar_buf);
    wrunlock();
    usleep(20000);  // 50Hz
}
```

Jetson 없이도 Qt 앱 테스트 가능.

### 2. 부하 테스트는 직접 루프

```c
// stress_test.c
// 최대한 빠르게 write해서 동기화 버티는지 확인
for (int i = 0; i < 1000000; i++) {
    wrlock();
    memcpy(shm->lidar_buf, data, size);
    wrunlock();
}
// → race condition, deadlock 없는지 확인
```
