### 코드 안에서 직접 측정

```c
// 송신 측: 타임스탬프 헤더에 박아서 보냄
typedef struct {
    uint64_t send_ts_us;  // 송신 시각
    uint32_t seq;         // 순서 (손실 감지)
    uint32_t payload_len;
} PacketHeader;

// 수신 측: 도착하자마자 차이 계산
uint64_t recv_ts = get_time_us();
uint64_t latency = recv_ts - hdr.send_ts_us;

// 시각 얻기
uint64_t get_time_us() {
    struct timespec ts;
    clock_gettime(CLOCK_MONOTONIC, &ts);
    return ts.tv_sec * 1000000 + ts.tv_nsec / 1000;
}
```

### 개선 전후 비교 구조

```c
experiment/
  baseline/
    metrics_20240701.csv   ← rwlock 구현 전
  rwlock/
    metrics_20240702.csv   ← rwlock 적용 후
  zerocopy/
    metrics_20240703.csv   ← zero-copy 적용 후
```

```python
# plot.py
import pandas as pd
import matplotlib.pyplot as plt

baseline = pd.read_csv('baseline/metrics.csv')
rwlock   = pd.read_csv('rwlock/metrics.csv')

plt.boxplot([baseline['latency_us'],
             rwlock['latency_us']],
            labels=['before', 'after'])
plt.ylabel('latency (us)')
plt.savefig('comparison.png')
```
