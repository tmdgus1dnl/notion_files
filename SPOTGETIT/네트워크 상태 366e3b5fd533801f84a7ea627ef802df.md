# 네트워크 상태

로봇 상태 카드의 네트워크 상태에 표시되는 xx ms는:

qRound(snapshot.linkRttMs)

입니다.

snapshot.linkRttMs는 Qt가 SHM에서 읽는 값이고:

snapshot.linkRttMs = state.link_rtt_ms;

즉 실제 원본은 bridge SHM의:

shm->state.link_rtt_ms

입니다.

이 값은 bridge에서 command ACK 패킷을 받았을 때만 이렇게 계산됩니다.

shm->state.link_rtt_ms =
(float)(shm->metrics.last_ack_us - ack->timestamp_us) / 1000.0f;

즉 표시되는 ms는 마지막 명령 ACK 기준 RTT 비슷한 값입니다.
ROS topic latency, global path UDP latency, 일반 UDP 수신 지연, ping RTT가 아닙니다.
