# 내일 할 작업

맵 상에서 시작, 끝 좌표를 클릭이나 터치로 설정 가능

끝 점까지 선택하면 오른쪽 하단에 경로 생성 버튼 생성 → 누르면 이미 받고 있는 global path 띄우기

시작이나 끝 점 이미 있는거 다시 클릭이나 터치하면 없어짐

경로 생성 된 이후 이동 버튼 누르면 로봇들 이동 시작(이거는 라파 → PC 통신 뚫어야함)

로봇 고장나서 추가투입버튼 누르고 1대 투입하면 이미 있는 시작, 끝 좌표로 경로 바로 생성

아마 고장난 로봇의 경로가 뜰거임. 이 상태에서 이동 누르면 이미 이동중인 로봇들은 그대로 가고 추가 투입 된 로봇만 이동 (이거는 라파 → 젯슨 통신 뚫어야함, CLASSIC pub? 할듯)

이미 경로 생성한 1번이 고장난걸 생성하는 쪽에서 어떻게 판단하고 있고 5번이 들어가면 1번이 가던 경로로 어떻게 다시 경로를 할당하는지 물어봐야함

긴급정지 구현..? 누르면 더미랑 로봇 둘다 바로 멈추게? → 이미 통신이 뚫려있다는 가정 하에.
